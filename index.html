Get Started

SyncStack is Open-Source

Syncstack is an open source database.

In-Memory SQL

You can select your in memory data with SQL syntax. Works with standard SQL, no weird dialects or new syntax to learn.

Easy to Use

You can create a table as you do in traditional db or json table. You can create a json table and query it.

Client libraries

Flexible messaging models with high-level APIs for Java, Go, C

Integrations

We support cloud native applications. We have Kubernetes and Docker integration. You can check all these at the documentation.
